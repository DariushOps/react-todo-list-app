# React Todo List App

A React application to manage projects and tasks with a clean and interactive interface.

## Features
- Create, view, and delete projects
- Add and remove tasks for each project
- Real-time updates using React state
- Responsive UI for desktop and mobile
- Modal dialogs for input validation and notifications

## Tech Stack
- React (Vite)
- useState for state management
- Functional components with props and refs
- Portal-based modal implementation
- CSS with Tailwind for styling

## Getting Started
### Install dependencies
```bash
npm install
```
## How It Works

- The sidebar lists all projects and allows adding new ones.

- Selecting a project displays its details and tasks.

- Users can add and remove tasks within each project.

- Input validation and feedback are handled via modals.
